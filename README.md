# golang开发的《暗黑破怀神2 Demo》  Diablo2_Demo

！转载请注明来源

### golang版本支持 1.18 1.19

1.18·版本可以手动修改下go mod里的golang版本

```golang

//1.18运行的情况下 把main.go的下面代码注释掉
debug.SetMemoryLimit(300 * 1024 * 1024)
```


## 游戏运行方法
```
1. git clone
2. go get 或 go mod vendor 
3. go run main.go  或  go build
```

## 控制说明

  1.键盘F键
    全局画面切换
  
  2.键盘Q键
    手机PC操作切换
    
  3.键盘D键
    显示Debug信息
       
  4.F1键
    野蛮人呐喊
    
  5.鼠标左键
    移动
    
  6.鼠标右键
    攻击
